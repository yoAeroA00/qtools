These bootloaders have only 1 patch - command 11 (EXEC), instead of it there used to be a dull plug of Power Off not Supported.
Team format:

11 00 <Up to 1020 bytes of the performed program>

Management is transferred to the most zero byte of the program in ARM mode. Input registers:

R0 - program launch address
R1 - response address

The program can fill up to 1020 bytes in a return buffer. Output registers:

R4 is the number of bytes placed in a return buffer.


The initial texts of the patches are in Pexec _*. Asm files
